# Инструкция по проверке работы приложения

## Проблема с запуском сервера

Если вы видите ошибку `Unknown option '--middlewares'`, это означает, что ваша версия json-server не поддерживает этот флаг напрямую.

## Решение

Я создал файл `server.js`, который запускает json-server программно с middleware.

### Шаг 1: Установите зависимости (если еще не установлены)

```bash
cd labs/K3344/Сокович Степан/lab3
npm install
```

### Шаг 2: Запустите JSON-сервер

```bash
npm run server
```

Теперь команда `npm run server` запустит `node server.js`, который правильно подключит middleware.

### Шаг 3: В другом терминале запустите Vue.js приложение

```bash
npm run dev
```

### Шаг 4: Откройте браузер

Перейдите на `http://localhost:5173`

## Альтернативный способ (если проблемы остаются)

Если `npm run server` все еще не работает, попробуйте:

```bash
node server.js
```

Или используйте старый способ без middleware (только для тестирования):

```bash
npx json-server --watch db.json --port 3000
```

Но в этом случае CORS и авторизация могут не работать правильно.
