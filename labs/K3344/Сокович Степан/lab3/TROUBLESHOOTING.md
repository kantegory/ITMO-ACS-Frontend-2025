# Решение проблем

## Проблема: Белый экран или ничего не отображается

### Проверка 1: Откройте консоль браузера (F12)

Если видите ошибки, они помогут понять проблему.

### Проверка 2: Убедитесь, что серверы запущены

**Терминал 1 - JSON Server:**
```bash
cd labs/K3344/Сокович Степан/lab3
npm run server
```

Должно быть:
```
JSON Server is running on http://localhost:3000
```

**Терминал 2 - Vue.js:**
```bash
cd labs/K3344/Сокович Степан/lab3
npm run dev
```

Должно быть:
```
VITE v5.x.x  ready in xxx ms
➜  Local:   http://localhost:5173/
```

### Проверка 3: Откройте правильный URL

Откройте: **http://localhost:5173** (не 3000!)

### Проверка 4: Проверьте, что зависимости установлены

```bash
cd labs/K3344/Сокович Степан/lab3
npm install
```

### Проверка 5: Очистите кэш и переустановите

```bash
rm -rf node_modules package-lock.json
npm install
npm run dev
```

## Частые ошибки

### Ошибка: "Cannot find module"
- Выполните `npm install`

### Ошибка: "Failed to fetch" или CORS
- Убедитесь, что JSON-server запущен на порту 3000
- Проверьте, что middleware.js подключен

### Ошибка: "Cannot read property of undefined"
- Откройте консоль браузера (F12) и посмотрите полную ошибку
- Проверьте, что все компоненты импортированы правильно

## Если ничего не помогло

1. Откройте консоль браузера (F12 → Console)
2. Скопируйте все ошибки
3. Проверьте Network tab - есть ли запросы к API?
4. Проверьте, что оба сервера запущены
