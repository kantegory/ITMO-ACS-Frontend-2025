# Музыкальный каталог оценки альбомов

## О проекте

В рамках 1 лабораторной работы разработан веб-сервис для просмотра, поиска и оценки музыкальных альбомов. Интерфейс реализован на HTML, CSS и Bootstrap с использованием JavaScript для интерактивных функций.

---

## Соответствие критериям задания

- Проект сверстан с помощью **HTML**, **CSS** и **Bootstrap**.
- Внедрены моменты, требующие **JavaScript**:  
    - Реализована динамическая фильтрация каталога (жанр, год, рейтинг).
    - Открытие карточек альбомов с подробной информацией.
    - (При необходимости легко добавить открытие модальных окон или всплывающих элементов).
- Все страницы соответствуют требуемому функционалу:
    - **Вход** (`login.html`)
    - **Регистрация** (`register.html`)
    - **Личный кабинет пользователя** (`profile.html`)
    - **Страница поиска с фильтрами** (`catalog.html`)

---

## Функционал

- Главная страница — каталог альбомов с фильтрами и адаптивным дизайном.
- Форма входа и регистрации с полями проверки.
- Личный кабинет пользователя — просмотр и изменение персональных данных, история активности.
- Возможность поиска альбомов и фильтрации по жанру, году, рейтингу.
- Красивые карточки альбомов с описанием, треками и рейтингом.
